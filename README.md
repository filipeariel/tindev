## Tecnoglogias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- ⚛️ **React Js**
- ⚛️ **React Native**
- 💹 **Node Js**
- 📄 **MongoDB**
- ♻️ **Socket IO**

## Projeto

O TinDev é uma plataforma criada para conectar devs de forma eficiente, utilizando a API do Github para cadastrar
usuário no aplicativo
